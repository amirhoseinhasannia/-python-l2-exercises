# BookBuddy

BookBuddy is a simple command-line (CLI) application that helps you track your reading habits and manage your personal library.  
You can register your books, log each reading session, and see how much progress you have made on every book.

## Features

- **Book registration**
  - Add different types of books:
    - Regular books
    - EBooks
    - AudioBooks

- **Reading logs**
  - Record each reading session with:
    - Book title
    - Amount read (pages or minutes)
    - Date and time
    - Optional notes or reflections

- **Progress tracking**
  - Calculate reading progress (percentage) for each book
  - See how much of a book you have read or listened to

- **Data management**
  - Save and load books and reading logs from files
  - Export data for reporting or backup

- **Simple CLI menu**
  - Text-based menu to:
    - Add new books
    - List all books
    - Log reading sessions
    - Show reading progress
    - Save and load data
